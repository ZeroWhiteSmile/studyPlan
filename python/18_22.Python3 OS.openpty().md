## Python3 OS.openpty()(报错module 'os' has no attribute 'openpty')

os.openpty() 方法用于打开一个新的伪终端对。返回 pty 和 tty的文件描述符。

```
os.openpty()
```

返回值 -- 返回文件描述符对，主从。

```
import os,sys

print(dir(os))

#主 pty, 从 tty
m, s = os.openpty()
print(m)
print(s)

#显示终端名
s = os.ttyname(s)
print(m)
print(s)
输出：报错module 'os' has no attribute 'openpty'
```
