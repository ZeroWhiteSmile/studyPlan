## Python3 OS.link()

os.link() 方法用于**创建硬链接**，名为参数 dst，指向参数 src。

该方法对于创建一个**已存在文件的拷贝**是非常有用的。

只支持在 Unix, Windows 下使用。

```
os.link(src, dst)
```

src -- 用于创建硬连接的源地址
dst -- 用于创建硬连接的目标地址
返回值 -- 该方法没有返回值。

```
import os,sys

path = 'test.txt'
#创建以上文件的拷贝
copyPath = 'test/test.txt'
os.link(path, copyPath)
```
