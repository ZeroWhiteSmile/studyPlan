## python3 os.dup2(fd, fd2)

os.dup2() 方法用于将一个文件描述符 fd 复制到另一个 fd2。

**Unix, Windows 上可用。**

```python
os.dup2(fd, fd2)
```

fd -- 要被复制的文件描述符
fd2 -- 复制的文件描述符
返回值 -- 没有返回值。

```python
import os,sys

with open('test.txt', 'a') as f:# a：追加模式
    # 将这个文件描述符代表的文件，传递给 1 描述符指向的文件（也就是 stdout）
    os.dup2(f.fileno(), 1)

    print('runoob')
```
