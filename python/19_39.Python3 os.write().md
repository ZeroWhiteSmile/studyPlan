## Python3 os.write()

os.write() 方法用于写入字符串到文件描述符 fd 中. 返回实际写入的字符串长度。

```python
os.write(fd, str)
```

fd -- 文件描述符。
str -- 写入的字符串。
返回值 -- 该方法返回写入的实际位数。

```python
# encoding: utf-8

import os, sys

# 打开文件
fd = os.open("f1.txt",os.O_RDWR|os.O_CREAT)

# 写入字符串
str = "This is runoob.com site"
ret = os.write(fd,bytes(str, 'UTF-8'))
#或者
ret = os.write(fd,str.encode())
print (ret)
```
