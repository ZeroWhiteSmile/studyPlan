## Python3 OS.read() 方法

os.read() 方法用于从文件描述符 fd 中读取最多 n 个字节，返回包含读取字节的字符串，文件描述符 fd对应文件已达到结尾, 返回一个空字符串。

在Unix，Windows中有效

```python
os.read(fd,n)
```

fd -- 文件描述符。
n -- 读取的字节。
返回值 -- 返回包含读取字节的字符串

os.O_RDWR : 以读写的方式打开

```python
import os,sys

#打开文件
fd = os.open("test.txt", os.O_RDWR)

#读取文本
ret = os.read(fd,12)
print(ret) #b'This is runo'

#关闭文件
os.close(fd)
```
