## markdown格式写法

## github上图片路径问题

* 必须使用./或绝对路径,不能使用../之类。

```
<img src='./img/xxx.png' />
```

## 插入代码片段

* 使用格式 ` ``` `加代码语言包裹代码

```git
git commit -m ''
```

```javascript
lat a = () => {}
```

```python
print("{} {}".format("hello", "world"))
```

```c
int main(int arg)
```

```Java
public static void main(String[]args){}
```
